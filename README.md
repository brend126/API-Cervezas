# Proyecto Tabla de Cervezas Artesanales

## Descripción

Este proyecto es una página web que muestra una tabla de cervezas artesanales obtenidas de una API pública. La tabla incluye información sobre el nombre, precio, valoración promedio, número de opiniones, y una imagen de cada cerveza. El objetivo es proporcionar una interfaz simple para visualizar y explorar una variedad de cervezas artesanales.

## Características

- **Visualización de cervezas:** Muestra una lista de cervezas artesanales con sus detalles relevantes.
- **API pública:** Los datos de las cervezas se obtienen de la API [Sample Beers API](https://api.sampleapis.com/beers/ale).
- **Interfaz responsiva:** La página está diseñada para ser visualizada correctamente en diferentes tamaños de pantalla.
- **Estilización personalizada:** Se utiliza CSS personalizado y Bootstrap para la presentación de la tabla.

## Tecnologías Utilizadas

- **HTML5**
- **CSS3**
  - Bootstrap 5
- **JavaScript**
  - Fetch API
- **API Pública:** [Sample Beers API](https://api.sampleapis.com/beers/ale)
